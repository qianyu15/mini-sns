# Mini SNS

Node.js + Express + SQLiteで作成した小型SNS。

## Features

- User registration
- Login
- Create posts
- Timeline display


## Tech Stack

Frontend:
- HTML
- CSS
- JavaScript

Backend:
- Node.js
- Express

Database:
- SQLite


## Run locally

Install dependencies:

```

npm install

```

Start:

```

npm start

```

Open:

```

[http://localhost:3000](http://localhost:3000)

```


## Deploy

Designed to run on Render.
